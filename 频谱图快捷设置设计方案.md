# 频谱图快捷设置设计方案

## 一、现状分析

### 1.1 当前架构
- **频谱图组件**: `DashboardHomeCard` - 主页仪表卡片
- **图表类型**: 频率频谱 / 时域波形 / 声级分布（Swiper切换）
- **全屏功能**: 右下角悬浮全屏按钮（⛶）进入横屏全屏模式
- **现有设置入口**: 
  - 时间加权/分析模式: `TimeWeightingDialog`
  - 频率加权: `FrequencyWeightingDialog`
  - 频谱显示设置: `SpectrumDisplaySettingsDialog`
  - 校准: 独立的校准流程组件

### 1.2 用户需求
在频谱图页面快速设置以下专业测量参数：
1. **升级校准** - 设备校准状态及快捷校准入口
2. **频率加权** - A/C/Z计权切换
3. **分析模式** - 快速/慢速/脉冲/自定义模式切换
4. **自定义模式配置** - FFT参数、平滑系数等高级设置

## 二、设计方案对比

### 方案A：浮动快捷工具栏（推荐 ⭐⭐⭐⭐⭐）

#### 设计思路
在频谱图区域添加智能浮动工具栏，采用"一键展开"的交互模式。

#### 具体实现
```
┌─────────────────────────────────────────┐
│  频率频谱图                              │
│                                         │
│    ┌───┐                                │
│    │ A │  ← 当前计权类型（可点击切换）     │
│    └───┘                                │
│         ┌───┐                           │
│         │快 │  ← 当前分析模式（可点击切换）  │
│         └───┘                           │
│              ┌───┐                      │
│              │ ⚙ │  ← 展开更多设置按钮     │
│              └───┘                      │
│                                         │
│                              ┌───┐      │
│                              │ ⛶ │      │  ← 现有全屏按钮
│                              └───┘      │
└─────────────────────────────────────────┘
```

#### 交互细节
1. **智能显示**: 工具栏默认收起为单个"⚙"按钮，3秒无操作自动收起
2. **点击展开**: 点击设置按钮展开为弧形/直线排列的功能图标
3. **状态指示**: 每个按钮显示当前设置状态（如"A"、"快"、"✓"）
4. **快捷修改**: 长按或二次点击直接进入对应设置弹窗

#### 功能按钮设计
| 按钮 | 图标/文字 | 显示状态 | 点击行为 |
|------|----------|----------|----------|
| 计权类型 | A/C/Z | 当前计权 | 弹出计权选择器 |
| 分析模式 | 快/慢/脉/自 | 当前模式 | 弹出分析模式选择器 |
| 校准状态 | ✓/! | 已校准/需校准 | 跳转校准或提示已校准 |
| 更多设置 | ⚙ | - | 展开完整设置菜单 |

#### 优势
- ✅ **空间高效**: 不占用频谱图显示区域
- ✅ **状态可见**: 实时显示当前关键参数
- ✅ **操作快捷**: 一键直达常用设置
- ✅ **专业感强**: 类似专业音频分析软件界面
- ✅ **扩展性好**: 后续可增加更多快捷按钮

---

### 方案B：底部状态栏方案（推荐 ⭐⭐⭐⭐）

#### 设计思路
在频谱图底部添加状态指示条，类似专业音频设备的参数显示栏。

#### 具体实现
```
┌─────────────────────────────────────────┐
│                                         │
│           频谱图显示区域                   │
│                                         │
│                                         │
├─────────────────────────────────────────┤
│ [A计权] [快速模式] [已校准] [设置 ▼]     │  ← 底部状态栏
└─────────────────────────────────────────┘
```

#### 交互细节
1. **常驻显示**: 底部状态栏始终可见
2. **点击修改**: 点击对应标签直接弹出选择器
3. **视觉层次**: 使用标签(Tag)样式，选中状态高亮

#### 优势
- ✅ **状态清晰**: 所有关键参数一目了然
- ✅ **操作直接**: 点击即改，无需展开
- ✅ **符合直觉**: 类似设备参数显示

#### 劣势
- ⚠️ **占用空间**: 需要24-32px高度
- ⚠️ **信息密度**: 参数多时可能拥挤

---

### 方案C：右键/长按菜单方案（推荐 ⭐⭐⭐）

#### 设计思路
在频谱图区域长按或右键呼出快捷菜单。

#### 交互细节
1. **长按触发**: 在频谱图区域长按0.5秒呼出菜单
2. **菜单内容**: 列出常用设置选项
3. **快捷操作**: 支持拖拽调整参数

#### 优势
- ✅ **零UI占用**: 不添加任何可见按钮
- ✅ **操作自然**: 符合移动端交互习惯

#### 劣势
- ⚠️ **发现性差**: 用户可能不知道此功能
- ⚠️ **操作门槛**: 需要学习成本

## 三、推荐方案：方案A - 浮动快捷工具栏

### 3.1 组件结构设计

```typescript
// 新增组件：SpectrumQuickSettings.ets
@ComponentV2
export struct SpectrumQuickSettings {
  @Param isExpanded: boolean = false;
  @Local pk: PreferenceKeys = PersistenceV2.connect(PreferenceKeys)!;
  
  // 快捷设置项配置
  private readonly quickSettings: QuickSettingItem[] = [
    { 
      type: 'weighting', 
      icon: null, // 使用文字 A/C/Z
      label: '计权',
      getValue: () => this.pk.weighting_type,
      onClick: () => this.showWeightingDialog()
    },
    { 
      type: 'analysis', 
      icon: null, // 使用文字 快/慢/脉/自
      label: '模式',
      getValue: () => this.getAnalysisModeShort(),
      onClick: () => this.showAnalysisModeDialog()
    },
    { 
      type: 'calibration', 
      icon: $r('app.media.ic_calibration'),
      label: '校准',
      getValue: () => this.pk.isCalibrated ? '✓' : '!',
      onClick: () => this.handleCalibrationClick()
    },
    { 
      type: 'custom', 
      icon: $r('app.media.ic_settings'),
      label: '自定义',
      onClick: () => this.showCustomConfigDialog()
    }
  ];
}
```

### 3.4 与现有功能整合

#### 复用现有Dialog
```typescript
// 点击计权按钮 -> 复用 FrequencyWeightingDialog
private showWeightingDialog() {
  this.weightingDialog = DialogHub.getSheet()
    .setContent(wrapBuilder(FrequencyWeightingDialogBuilder), {
      currentWeighting: this.pk.weighting_type,
      onClose: () => this.closeWeightingDialog(),
      onSettingsApplied: (weighting) => {
        // 应用新设置，更新按钮状态
        this.pk.weighting_type = weighting;
      }
    })
    .build();
}

// 点击分析模式按钮 -> 复用 TimeWeightingDialog
private showAnalysisModeDialog() {
  this.analysisDialog = DialogHub.getSheet()
    .setContent(wrapBuilder(TimeWeightingDialogBuilder), {
      currentMode: this.pk.audio_analysis_mode,
      onClose: () => this.closeAnalysisDialog(),
      onSettingsApplied: (mode) => {
        this.pk.audio_analysis_mode = mode;
      }
    })
    .build();
}
```

#### 自定义模式特殊处理
```typescript
// 如果当前是自定义模式，显示额外的配置入口
if (this.pk.audio_analysis_mode === AudioAnalysisMode.CUSTOM_TIME_WEIGHTING) {
  // 显示自定义配置子菜单
  this.showCustomConfigSubmenu();
}
```

## 七、总结

本方案在保持原有功能完整性的前提下，通过**浮动快捷工具栏**的设计，实现了专业测量参数的快速访问：

1. **功能复用**: 完全复用现有的 `TimeWeightingDialog`、`FrequencyWeightingDialog` 等成熟组件
2. **最小侵入**: 仅在 `DashboardHomeCard` 中添加一个组件，不影响其他代码
3. **用户体验**: 状态可见、操作快捷、符合专业软件交互习惯
4. **扩展性强**: 预留了场景预设、手势操作等扩展空间

建议按照P0→P1→P2→P3的顺序逐步实现，优先完成核心快捷设置功能。
