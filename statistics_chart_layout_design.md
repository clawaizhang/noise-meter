# 统计分布图布局优化设计方案

## 当前问题分析
1. **布局重叠**：统计指标与直方图在同一区域绘制，导致内容重叠
2. **空间利用不合理**：小屏幕下空间紧张，大屏幕下空间浪费
3. **视觉层次混乱**：缺乏明确的视觉层次和信息优先级

## 重新设计的布局策略

### 方案一：垂直分离布局（推荐）
```
┌─────────────────────────────────┐
│          图表标题               │
├─────────────────────────────────┤
│                                 │
│          直方图区域             │
│        (主要可视化内容)         │
│                                 │
├─────────────────────────────────┤
│  统计指标区域 (关键数据展示)    │
│  平均值 | 最大值 | 最小值 | 标准差 │
└─────────────────────────────────┘
```

### 方案二：侧边布局（大屏幕专用）
```
┌─────────────────────────────────┐
│          图表标题               │
├─────────────┬───────────────────┤
│             │                   │
│  统计指标   │    直方图区域     │
│  区域       │                   │
│             │                   │
└─────────────┴───────────────────┘
```

## 响应式布局策略

### 小屏幕布局 (宽度 ≤ 360px)
- **布局方式**：垂直堆叠
- **Canvas尺寸**：宽度占满，高度180px
- **统计指标位置**：图表下方，水平排列关键指标
- **字体大小**：标题16px，内容12px

### 中等屏幕布局 (宽度 361px - 600px)  
- **布局方式**：垂直分离
- **Canvas尺寸**：固定宽度280px，高度220px
- **统计指标位置**：图表下方，两列布局
- **字体大小**：标题18px，内容14px

### 大屏幕布局 (宽度 > 600px)
- **布局方式**：侧边布局
- **Canvas尺寸**：固定宽度320px，高度240px
- **统计指标位置**：图表左侧，垂直排列
- **字体大小**：标题20px，内容16px

## 具体实现方案

### 1. 重新组织绘制顺序
- 先绘制背景和标题
- 再绘制直方图（主要可视化内容）
- 最后绘制统计指标（次要信息）

### 2. 统计指标布局优化
- **小屏幕**：水平排列关键指标（平均值、最大值、最小值）
- **中等屏幕**：两列布局（平均值/最大值，最小值/标准差）
- **大屏幕**：垂直完整列表

### 3. 直方图优化
- 动态调整柱子间距和宽度
- 优化计数标签位置，避免重叠
- 分贝区间标签位置优化

### 4. 视觉层次设计
- **一级信息**：直方图（主要可视化）
- **二级信息**：关键统计指标（平均值、最大值）
- **三级信息**：详细统计指标（最小值、标准差、时长）

## 技术实现要点

1. **响应式边距计算**：根据屏幕尺寸动态计算边距
2. **字体大小自适应**：基于屏幕尺寸调整字体大小
3. **布局切换逻辑**：根据断点切换不同的布局策略
4. **Canvas尺寸优化**：保持合适的宽高比，避免变形

## 布局优化实施总结

### 已实施的优化措施

#### 1. 垂直间距压缩
- **顶部边距**：从 45-50 减少到 25-35（减少 40-45%）
- **底部边距**：从 80-90 减少到 50-65（减少 35-40%）
- **统计指标位置**：从 +50vp 减少到 +25vp（减少 50%）

#### 2. 画布尺寸优化
- **小屏幕高度**：从 240px 减少到 180px（减少 25%）
- **中等屏幕高度**：从 260px 减少到 200px（减少 23%）
- **全屏模式**：动态计算，最大 220px

#### 3. 统计指标布局密度提升
- **行间距**：从 20-40vp 减少到 12-30vp（减少 40-50%）
- **紧凑排列**：根据屏幕尺寸智能选择布局方式
- **字体优化**：适当减小字体大小提高密度

#### 4. 坐标轴标签优化
- **分贝区间标签**：从 +15vp 减少到 +10vp（减少 33%）
- **X轴标签**：从 +35vp 减少到 +25vp（减少 29%）
- **计数标签**：从 +6vp 减少到 +4vp（减少 33%）

### 预期效果
- 消除所有重叠问题
- 提高信息可读性
- 优化空间利用率（整体减少 30-40% 垂直空白）
- 保持视觉一致性
- 全屏模式下充分利用屏幕空间
