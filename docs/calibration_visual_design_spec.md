# 声级校准界面视觉设计规范

## 设计原则

### 简洁精美
- 保持界面清爽，避免信息过载
- 使用适当的留白和间距
- 采用现代化卡片设计
- 统一的视觉语言

### 专业可信
- 清晰的声学专业内容
- 精确的数值显示
- 可靠的状态指示
- 专业的色彩搭配

### 交互友好
- 直观的操作反馈
- 流畅的动画效果
- 智能的提示系统
- 响应式布局适配

## 色彩方案

### 主色调
- **主要操作色**：`#007AFF` (系统蓝色)
- **成功状态**：`#34C759` (绿色)
- **警告状态**：`#FF9500` (橙色)
- **错误状态**：`#FF3B30` (红色)
- **中性背景**：`#F2F2F7` (浅灰)

### 状态色彩编码
- **校准中**：蓝色渐变动态效果
- **校准完成**：稳定绿色指示
- **需要校准**：提示橙色
- **校准错误**：警示红色
- **环境良好**：淡绿色背景
- **环境嘈杂**：淡橙色背景

## 布局结构

### 界面层次
```
┌─────────────────────────────────┐
│  核心校准控制区 (固定位置)       │
├─────────────────────────────────┤

└─────────────────────────────────┘
```

### 组件间距规范
- **卡片间距**：16px
- **内部元素间距**：12px
- **标题与内容间距**：8px
- **按钮间距**：8px

## 动画效果规范

### 微交互动画

**按钮点击反馈**：
```typescript
// 点击缩放效果
Button().scale({ 
  x: 0.95, 
  y: 0.95 
}).animation({
  duration: 100,
  curve: curves.easeOut
})
```

**状态切换动画**：
```typescript
// 淡入淡出过渡
Component().opacity(0).animation({
  duration: 300,
  curve: curves.easeInOut
})
```

### 进度动画

**环形进度条**：
```typescript
// 进度环动画
ProgressRing()
  .value(progress)
  .animation({
    duration: 500,
    curve: curves.springMotion()
  })
```

**数值变化动画**：
```typescript
// 平滑数值过渡
Text(value)
  .animation({
    duration: 200,
    curve: curves.easeOut
  })
```

### 内容展开动画

**面板展开**：
```typescript
// 高度变化动画
Panel()
  .height(expanded ? 200 : 0)
  .animation({
    duration: 400,
    curve: curves.springMotion(0.6, 0.8)
  })
```

## 组件设计规范

### 1. 增强型滑动条

**视觉设计**：
- 轨道高度：24px
- 滑块大小：24×24px
- 预设点标记：小圆点指示
- 数值提示：悬浮显示

**交互反馈**：
- 滑块拖拽时的缩放效果
- 预设点吸附的轻微震动
- 数值变化的平滑过渡

### 2. 自动校准按钮

**状态设计**：
- **默认状态**：蓝色填充按钮
- **校准中**：动态进度环
- **完成状态**：绿色成功图标
- **错误状态**：红色警示图标

**动画序列**：
1. 点击后开始3秒倒计时
2. 显示环形进度动画
3. 实时环境检测反馈
4. 完成时的庆祝动画

### 3. 状态指示器

**环形进度指示**：
- 外环：校准进度
- 内环：环境噪声水平
- 中心：当前声级数值
- 颜色：根据状态变化

**实时波形图**：
- 显示最近5秒的声级变化
- 平滑的波形动画
- 颜色编码表示不同强度
- 可交互的时间轴

### 4. 专业内容面板

**展开/收起机制**：
- 默认收起状态
- 点击标题区域展开
- 平滑的高度动画
- 保持滚动位置

**内容组织**：
- 技术参数表格形式
- 声学原理分点说明
- 应用场景卡片展示
- 交互式图表显示

## 交互状态设计

### 校准过程状态

**准备阶段**：
- 显示环境检测结果
- 提供校准建议
- 指导用户操作

**进行阶段**：
- 实时进度显示
- 环境变化监控
- 异常状态提示

**完成阶段**：
- 成功/失败状态显示
- 校准结果总结
- 后续操作建议

### 错误处理状态

**环境嘈杂**：
- 提示寻找安静环境
- 显示当前噪声水平
- 提供重试建议

**设备问题**：
- 检测麦克风状态
- 提供故障排除指导
- 建议设备检查

**校准失败**：
- 分析失败原因
- 提供解决方案
- 记录错误日志

## 视觉一致性

### 图标设计
- 统一使用系统图标库
- 保持一致的线条粗细
- 统一的圆角风格
- 协调的色彩搭配

### 字体规范
- 标题：系统字体，中等粗细
- 正文：系统字体，常规粗细
- 数值：等宽字体，确保对齐
- 标签：较小字号，次要颜色

### 阴影效果
- 卡片阴影：轻微投影
- 按钮阴影：点击时消失
- 浮动元素：适当阴影增强层次
- 背景元素：无阴影保持简洁

## 性能优化考虑

### 动画性能
- 避免过多同时运行的动画
- 使用硬件加速的动画属性
- 合理设置动画时长
- 预加载必要的资源

### 内存管理
- 及时清理不再使用的组件
- 优化图片和资源大小
- 合理使用缓存策略
- 监控内存使用情况

## 可访问性设计

### 视觉可访问性
- 足够的颜色对比度
- 清晰的字体大小
- 重要的状态使用多种提示方式
- 支持系统字体大小设置

### 操作可访问性
- 足够的触摸目标大小
- 清晰的焦点指示
- 键盘导航支持
- 语音提示兼容
