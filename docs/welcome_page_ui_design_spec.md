# 欢迎页UI设计规范

## 整体设计风格
- **设计语言**：简洁现代，专业感强
- **配色方案**：基于项目现有配色系统
- **字体系统**：使用DesignConstants规范
- **动画效果**：适度使用，增强用户体验

## 页面指示器设计

### 位置和样式
```
底部居中显示：
● ○ ○ ○   (第1页)
○ ● ○ ○   (第2页)  
○ ○ ● ○   (第3页)
○ ○ ○ ●   (第4页)
```

### 设计细节
- **激活状态**：实心圆，蓝色 (#1890FF)，直径8px
- **非激活状态**：空心圆，浅灰色 (#E0E0E0)，直径8px，透明度0.5
- **间距**：圆点之间间距8px
- **位置**：距离底部40px

## 动画效果设计

### 页面切换动画
- **滑动效果**：页面水平滑动，时长300ms，缓动曲线
- **淡入淡出**：新页面淡入，旧页面淡出

### 组件级动画
- **第1页**：应用图标轻微缩放动画（1.0 → 1.05 → 1.0）
- **第2页**：分贝值平滑变化，进度条动态填充
- **第3页**：频谱波浪连续移动
- **第4页**：权限卡片轻微悬停效果

## 响应式设计

### 手机竖屏
- 内容垂直居中
- 按钮宽度80%-100%
- 图标尺寸适中

### 手机横屏
- 内容适当调整间距
- 保持可读性

### 平板
- 增大图标和字体尺寸
- 调整布局比例
