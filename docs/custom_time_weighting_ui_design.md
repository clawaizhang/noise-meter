## 界面切换逻辑

### 状态1：自定义模式关闭
- **显示内容**：
  - 快速、慢速、脉冲三个标准模式选项卡
  - 实时预览面板
  - 适用场景展示
  - 可折叠的技术参数、声学原理、窗函数设置区域

- **隐藏内容**：
  - 自定义配置面板

### 状态2：自定义模式开启
- **显示内容**：
  - 自定义配置面板（包含所有可配置参数）
  - 技术参数显示（实时更新）

- **隐藏内容**：
  - 快速、慢速、脉冲模式选项卡
  - 实时预览面板
  - 适用场景展示
  - 声学原理区域
  - 窗函数设置区域

## 自定义配置参数设计

### 1. FFT大小设置
- **类型**：下拉选择框
- **选项**：1024, 2048, 4096, 8192
- **默认值**：2048
- **说明**：FFT大小影响频率分辨率

### 2. 更新间隔设置
- **类型**：下拉选择框
- **选项**：35ms, 100ms, 200ms, 500ms, 1000ms
- **默认值**：200ms
- **说明**：数据更新频率

### 3. 窗函数类型
- **类型**：下拉选择框
- **选项**：汉宁窗, 布莱克曼窗
- **默认值**：汉宁窗
- **说明**：影响频谱分析的频率分辨率和旁瓣抑制

### 4. 平滑因子
- **类型**：滑块或下拉选择框
- **范围**：0.05 - 0.5
- **步长**：0.05
- **默认值**：0.2
- **说明**：控制数据平滑程度

### 5. 重叠率
- **类型**：滑块或下拉选择框
- **范围**：0.1 - 0.9
- **步长**：0.05
- **默认值**：0.4
- **说明**：FFT窗口重叠比例

## 交互流程

### 开启自定义模式
1. 用户点击"自定义模式"开关
2. 界面立即切换，隐藏标准模式内容
3. 显示自定义配置面板
4. 技术参数区域显示当前自定义配置

### 配置参数
1. 用户调整任意配置参数
2. 技术参数区域实时更新显示
3. 配置立即生效，无需额外确认

### 保存配置
1. 用户点击"保存配置"按钮
2. 当前配置保存到应用存储
3. 下次开启自定义模式时自动加载

### 重置默认
1. 用户点击"重置默认"按钮
2. 配置恢复到默认值
3. 技术参数区域更新显示

### 关闭自定义模式
1. 用户再次点击"自定义模式"开关
2. 界面切换回标准模式
3. 恢复显示快速、慢速、脉冲模式选择

## 视觉设计要点

### 1. 自定义开关位置
- 位于模式选择区域上方
- 使用明显的开关控件
- 提供清晰的"自定义模式"标签

### 2. 配置面板设计
- 使用卡片式布局
- 清晰的参数分组
- 直观的下拉选择控件
- 操作按钮位于面板底部

### 3. 状态切换动画
- 平滑的显示/隐藏过渡
- 保持界面响应性
- 避免闪烁或跳动

### 4. 技术参数显示
- 使用列表形式清晰展示
- 实时更新参数值
- 提供参数说明提示

## 技术实现考虑

### 1. 状态管理
- 使用@Local变量管理自定义模式开关状态
- 实时响应配置参数变化
- 确保界面状态一致性

### 2. 配置存储
- 使用AppStorageV2存储自定义配置
- 支持配置的持久化
- 提供默认配置值

### 3. 性能优化
- 配置变化时立即更新显示
- 避免不必要的重渲染
- 利用现有的窗函数预缓存机制
