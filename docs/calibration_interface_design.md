## 声学专业内容库

### 1. 校准原理说明

**声级校准基本原理**：
- 声压级(SPL)与电信号转换
- 麦克风灵敏度补偿
- 环境噪声基准校正
- 频率响应均衡

**技术参数定义**：
- 校准精度：±0.5dB
- 频率范围：20Hz-20kHz
- 动态范围：30-130dB
- 采样率：48kHz

### 2. 声压级计算方式

**计算公式**：
```
SPL = 20 × log10(P / P0)
其中：
P = 测量声压
P0 = 参考声压 (20μPa)
```

**校准补偿**：
```
校准值 = 参考声级 - 测量声级
最终声级 = 原始声级 + 校准值
```

### 3. 麦克风灵敏度调整

**灵敏度参数**：
- 标准灵敏度：-26dBFS/Pa
- 实际灵敏度：设备特定
- 补偿系数：根据设备调整

**校准流程**：
1. 在安静环境中测量基准噪声
2. 计算与标准值的差异
3. 应用补偿系数
4. 验证校准精度

### 4. 环境噪声补偿算法

**补偿策略**：
- 实时环境噪声监测
- 自适应滤波算法
- 频率加权补偿
- 时间加权平均
