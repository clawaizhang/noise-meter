# 自动保存功能界面设计

## 界面布局设计

### 1. 设置页面入口

**位置**：在设置页面的"音频记录"和"后台运行"设置之间

**布局**：
```
┌─────────────────────────────────────────┐
│ [保存图标] 自动保存设置                 │
│         每5分钟自动保存（综合应用） →   │
└─────────────────────────────────────────┘
```

**状态显示逻辑**：
- 禁用状态：`自动保存已禁用`
- 启用状态：`每{间隔}分钟自动保存（{场景描述}）`

### 2. 自动保存设置弹框

#### 整体布局
```
┌─────────────────────────────────────────┐
│  自动保存设置                           │
├─────────────────────────────────────────┤
│  □ 启用自动保存                         │
│                                         │
│  ┌─────────┐ ┌─────────┐ ┌─────────┐    │
│  │  3分钟  │ │  5分钟  │ │ 10分钟  │    │
│  │ 工业监测│ │ 综合应用│ │社区投诉 │    │
│  └─────────┘ └─────────┘ └─────────┘    │
│                                         │
│  ┌─────────┐                            │
│  │ 20分钟  │  自定义间隔：[ 5 ] 分钟    │
│  │健康监测 │   （1-60分钟）             │
│  └─────────┘                            │
│                                         │
│  ┌─────────┐     ┌─────────┐            │
│  │   取消   │     │   确认   │            │
│  └─────────┘     └─────────┘            │
└─────────────────────────────────────────┘
```

#### 详细设计

**开关区域**：
- 使用标准的 Toggle 开关组件
- 开关状态控制下方选项的显示/隐藏

**预设选项区域**：
- 4个预设按钮，每个包含：
  - 时间标签（3分钟/5分钟/10分钟/20分钟）
  - 场景描述（工业监测/综合应用/社区投诉/健康监测）
- 按钮样式：选中状态使用强调色，未选中使用默认色

**自定义输入区域**：
- 数字输入框，范围限制1-60
- 输入时自动切换到自定义模式
- 实时验证输入范围

**按钮区域**：
- 确认按钮：应用设置并关闭弹框
- 取消按钮：放弃更改并关闭弹框

### 3. 交互逻辑

#### 开关控制
- 关闭状态：隐藏所有保存间隔选项
- 开启状态：显示保存间隔选项

#### 选项选择
- 点击预设按钮：自动选中并更新自定义输入框
- 自定义输入：自动切换到自定义模式
- 实时预览：显示当前选择的间隔和描述

#### 数据同步
- 弹框内部直接操作 PreferenceKeys
- 设置立即生效，无需回调参数
- 关闭弹框时自动保存设置

### 4. 视觉设计

#### 颜色方案
- 开关开启：绿色主题
- 选中状态：强调色（如蓝色）
- 未选中状态：默认灰色
- 禁用状态：浅灰色

#### 图标使用
- 设置入口：使用现有的保存图标 `$r('app.media.ic_save')`
- 弹框标题：使用相同的保存图标

#### 间距和尺寸
- 遵循现有的 DesignConstants
- 按钮高度：56px
- 选项卡片：统一尺寸
- 内外边距：使用标准间距值

### 5. 实现要点

#### 弹框组件结构
```typescript
@ComponentV2
export struct AutoSaveSettingsDialog {
  @Local pk: PreferenceKeys = PersistenceV2.connect(PreferenceKeys)!;
  @Local enabled: boolean = this.pk.auto_save_enabled;
  @Local interval: number = this.pk.auto_save_interval;
  @Local selectedPreset: number = 1; // 默认选中5分钟
  
  build() {
    Column() {
      // 开关区域
      this.buildToggleSection()
      
      if (this.enabled) {
        // 预设选项区域
        this.buildPresetOptions()
        
        // 自定义输入区域
        this.buildCustomInput()
      }
      
      // 按钮区域
      this.buildActionButtons()
    }
  }
}
```

#### 状态管理
- 弹框内部维护临时状态
- 确认时同步到 PreferenceKeys
- 取消时丢弃临时状态

### 6. 用户体验优化

#### 状态反馈
- 选中状态的视觉突出
- 输入验证的即时反馈
- 保存成功的提示信息

#### 无障碍访问
- 为所有交互元素添加 accessibilityDescription
- 支持键盘导航
- 屏幕阅读器友好

这个设计保持了界面的简洁性，同时提供了清晰的选项描述，让用户能够根据实际需求选择合适的自动保存间隔。
