# 噪音检测报告 V2 设计规范

## 一、设计目标

1. **提升专业性** - 让报告看起来更像专业的检测报告
2. **优化信息架构** - 信息层级清晰，阅读流畅
3. **增强可读性** - 数据可视化优化，重点突出
4. **改善用户体验** - 增加健康评估、标准对比等实用功能

## 二、信息架构重构

### 原架构问题
```
[BasicInfo] 平铺展示，无重点
[Statistics] 描述过长，分散展示
[NoiseChart] 深色背景突兀
```

### 新架构
```
[ReportHeader]      报告头部 + 综合评分（核心信息）
[DataOverview]      四个关键指标卡片（数据概览）
[NoiseChartV2]      噪音趋势 + 标准对比线
[HealthImpact]      健康影响评估（新增）
[NoiseDistribution] 分布统计 + 智能分析
[BasicInfoV2]       基本信息 + 备注编辑
```

## 四、组件设计说明

### 1. ReportHeader 报告头部

**功能：**
- 展示报告元信息（编号、时间、时长）
- 计算并展示综合评分
- 提供分享、关闭操作

**设计要点：**
- 大字号展示综合评分，一目了然
- 评分用颜色区分等级
- 操作按钮使用圆形图标按钮

### 2. DataOverview 数据概览

**功能：**
- 四个关键指标并列展示
- 最小值、平均值、最大值、波动率

**设计要点：**
- 使用网格布局，四个指标等宽
- 平均值卡片突出显示（加边框）
- 每个指标配颜色标签

### 3. NoiseChartV2 趋势图

**改进点：**
- 浅色主题，与整体协调
- 添加国家标准参考线
- 使用贝塞尔曲线平滑数据
- 峰值点特殊标记
- 渐变填充增强视觉效果

### 4. HealthImpactAssessment 健康评估

**新增功能：**
- 风险等级评估
- 健康影响说明
- 防护建议措施
- 国家标准对比表格

**设计要点：**
- 风险等级用大图标+颜色区分
- 建议措施用勾选列表形式
- 标准对比可折叠

### 5. NoiseDistribution 分布统计

**改进点：**
- 智能生成分析结论
- 可视化条形图展示分布
- 添加数据摘要（采样点、高噪时长、稳定度）

### 6. BasicInfoV2 基本信息

**改进点：**
- 网格布局展示时间信息
- 备注编辑入口更明显
- 空备注状态显示添加按钮

## 五、交互设计

### 1. 页面加载
- 滚动进入，带弹簧效果
- 各模块依次淡入

### 2. 图表交互
- 支持缩放查看（未来扩展）
- 点击数据点显示详情（未来扩展）

### 3. 备注编辑
- 点击备注区域弹出编辑弹窗
- 实时保存，Toast提示

### 4. 分享功能
- 生成格式化的分享文本
- 支持多平台分享（未来扩展）

## 六、响应式设计

### 适配规则
- 移动端优先设计
- 平板端适当增宽边距
- 大字号模式支持

## 七、无障碍设计

- 重要信息不仅依赖颜色
- 图标配合文字说明
- 支持屏幕阅读器

## 八、实现注意事项

1. **颜色定义** - 需要在资源文件中定义所有颜色
2. **图标资源** - 需要添加新的图标资源
3. **图表性能** - Canvas绘制注意性能优化
4. **数据计算** - 波动率、稳定度等计算在Computed中完成
5. **错误处理** - 空数据状态要有友好提示

## 九、与原版的对比

| 维度 | 原版 | V2新版 |
|------|------|--------|
| 信息层级 | 平铺，无重点 | 分层，有主次 |
| 视觉风格 | 系统默认 | 专业报告风格 |
| 数据可视化 | 深色图表 | 浅色+标准线 |
| 健康评估 | 无 | 有 |
| 标准对比 | 无 | 有 |
| 分享功能 | 无 | 有 |
| 智能分析 | 简单 | 详细 |

## 十、后续优化方向

3. **地图集成** - 检测位置地图展示
4. **语音备注** - 支持语音备注
5. **云同步** - 报告云端备份
